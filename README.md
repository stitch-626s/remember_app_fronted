# Remember App（前端部分）

> Remember App 是一个面向大规模题库学习的练题管理系统，支持用户高效地进行题目练习与学习。

Remember App 前端采用 Vue 3 + Vite 构建，提供流畅的交互体验与完善的学习功能。

## 技术栈

- Vue 3.5.30
- Vite 8.0.0
- Node.js 20.19.0

## 快速开始

### 准备环境

确保已安装 Node.js 20.19.0 或更高版本。

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

### 生产环境构建

```bash
npm run build
```

构建产物输出至 `dist` 目录。
